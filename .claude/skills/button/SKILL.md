# Кнопки ArrowDS

## Варианты

| Вариант | Reference | Когда использовать |
|---|---|---|
| **Primary** | `references/button-primary.md` ✅ | Главный CTA — одно главное действие на экране/секции |
| Secondary | *(в разработке)* | Второстепенное действие рядом с Primary |
| Tertiary | *(в разработке)* | Третичное, менее заметное |
| Ghost | *(в разработке)* | Текстовая кнопка без фона, минимальный визуальный вес |
| Addition | *(в разработке)* | Финальные действия: корзина, оплата |
| Contrast | *(в разработке)* | Тёмная кнопка / инверсия на светлом фоне |
| Clean | *(в разработке)* | Белая кнопка для цветных секций |

## Размерные модификаторы

Добавь к кнопке класс `.btn--{size}`:

| Класс | Высота | Размер шрифта | Когда |
|---|---|---|---|
| `btn--600` | 52px | 16px | Крупные CTA, hero-секции |
| `btn--500` | 48px | 16px | Главные действия в блоках |
| `btn--400` | 40px | 14px | Стандартный размер (default) |
| `btn--300` | 36px | 14px | Компактные блоки |
| `btn--200` | 32px | 13px | Карточки, плотные списки |
| `btn--100` | 24px | 13px | Inline-действия |
| `btn--50` | 20px | 12px | Самые компактные контролы |

## CSS-файлы

| Вариант | CSS | Что включает |
|---|---|---|
| Primary | `references/button-primary.css` | `.btn` base + все размеры + `.btn-primary` |

## Алгоритм

1. Определи вариант (обычно Primary если один CTA, Secondary если рядом с Primary)
2. Прочитай reference MD-файл варианта (HTML-разметка и описание состояний)
3. Подключи соответствующий CSS-файл из таблицы выше (один раз в глобальный CSS)
4. Добавь размерный модификатор `.btn--{size}` по контексту (если не указан — используй `btn--400`)
